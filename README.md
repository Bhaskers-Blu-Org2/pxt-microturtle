# micro turtle

A LOGO-like turtle library for the micro:bit in MakeCode.

```blocks
turtle.setPosition(0, 0)
turtle.setSpeed(45)
basic.forever(() => {
    turtle.forward(4)
    turtle.turnLeft()
})
```
